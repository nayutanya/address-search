# 住所検索

郵便番号を入力して住所を検索できるWebサイトです。

## 使用技術

- TypeScript
- Vite
- Sass (SCSS)
- Swiper.js
- ESLint
- Prettier

## 必要環境

- Node.js 24.14.0

## セットアップ手順

1. リポジトリをクローン
git clone https://github.com/nayutanya/address-search.git
cd address-search

2. パッケージをインストール
npm install

3. 開発サーバを起動
npm run dev

4. ブラウザで確認
http://localhost:5174 を開く

## コマンド一覧

| コマンド | 内容 |
|---|---|
| `npm run dev` | 開発サーバを起動 |
| `npm run build` | ビルド |
| `npm run lint` | ESLintを実行 |
| `npm run format` | Prettierを実行 |

## 動作確認ブラウザ

- Chrome
- Safari

## 対応デバイス

- PC
- モバイル（iOS・Android）
